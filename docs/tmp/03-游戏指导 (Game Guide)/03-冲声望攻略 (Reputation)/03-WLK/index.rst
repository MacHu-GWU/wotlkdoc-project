巫妖王之怒声望攻略
===============================================================================
**简要说明**

在巫妖王之怒版本中, 比较重要的声望有:

- **联盟先遣军 / 部落远征军**, PVP 头部附魔. 联盟先遣军的军需官在北风苔原的港口和嚎风峡湾的港口. 部落远征军的军需官在北风苔原的要塞和嚎风峡湾的港口.
- **祈伦托**, 任务主要在 北风苔原 / 达拉然 地图, 提供 法系 DPS 职业头部附魔, 军需官在达拉然内.
- **龙眠协调者**, 任务主要在龙骨荒野地图, 提供治疗职业头部附魔, 军需官在龙骨荒野的龙眠神殿顶部.
- **银白十字军**, 任务主要在 祖尔德拉克 / 寒冰皇冠 地图, 提供 Tank 职业头部附魔, 军需官在寒冰皇冠右下角的营地.
- **黯刃骑士团**, 任务主要在 祖尔德拉克 / 寒冰皇冠 地图, 提供 物理 DPS 职业头部附魔, 军需官需要任务和声望解锁, 在寒冰皇冠左上角的营地.
- **霍迪尔之子**, 各大职业肩部附魔, 需要任务解锁, 军需官在风暴群山地图右下的营地.

**五大头部附魔声望** 的冲法非常简单, 找军需官购买战袍之后, 穿上战袍打任何 5 人副本都能提升对应阵营的声望, 相比TBC, 不知道方便了多少. 但是在崇敬之后, 就只有刷英雄副本才有声望了. 并且, 各大声望都有一定数量的日常任务可接, 也可以额外提升声望.

**霍迪尔之子声望** 需要在达拉然的下水道的地精处接到引导任务之后 (是一个要你去风暴群山的 K3 营地的跑腿任务), 前往风暴群山的 K3 营地, 做完一个长达 50 个任务左右的任务链, 才能最终开启霍迪尔之子声望. 和其他声望不同的是, 霍迪尔之子没有战袍, 无法通过刷副本的方法提升, 只能够做每日六个日常任务提升声望. 所以到达 80 级之后, 每日的霍迪尔之子日常任务绝不能错过.

**狂心氏族** 和 **神谕者** 是在索拉查盆地的次要的声望, 需要做任务链开启. 两个声望相互仇恨, 你只能选择一个. 奖励主要是一些小玩具, 属于可选声望.
