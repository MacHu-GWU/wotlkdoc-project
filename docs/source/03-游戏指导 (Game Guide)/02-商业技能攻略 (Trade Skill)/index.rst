商业技能攻略
==============================================================================

商业技能玩家主要关心这样几个问题:

1. 如何能多快好省地将技能冲满?
2. 特定的配方如何获取?
3. 有哪些制品, 如何运作能给你带来利润?

对于问题1, :ref:`TradeskillInfo` 能告诉你哪些配方适合用来冲技能. 对于问题2, :ref:`AckisRecipeList` 能告诉你所有未学会的配方以及如何获得. 对于问题3, 那就是仁者见仁智者见智了.

特殊材料掉落说明
------------------------------------------------------------------------------

- ``冰冻符文``: 60级Naxx掉落, 是冰抗套装的材料.
- ``源生虚空``: TBC5人英雄副本最后一个Boss必掉.
- ``太阳之尘``: 太阳井高地副本小怪掉落.
- ``冰冻宝珠``: WLK5人英雄副本最后一个Boss必掉, 可以在达拉然处用牌子兑换.
- ``符文宝珠``: 奥杜尔10H和25人副本下Boss掉落, 可以在达拉然处用牌子兑换.
- ``十字军宝珠``: 十字军试炼10H和25人副本下Boss掉落, 可以在达拉然处用牌子兑换.
- ``源生萨隆邪铁``: 冰冠城塞10H和25人副本下Boss掉落, 可以在达拉然处用牌子兑换.
