木喉熊怪
==============================================================================


简介
------------------------------------------------------------------------------
木喉熊怪是位于费伍德森林北部通往冬泉谷的洞穴中的中立NPC。双手武器25敏和单手武器15敏在经典旧世是盗贼和猎人趋之若鹜的两个附魔。


声望奖励
------------------------------------------------------------------------------

- 附魔: 双手武器25敏捷
- 附魔: 单手武器15敏捷


冲声望的方法
------------------------------------------------------------------------------


Farmer方法
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

- 在费伍德森林杀死熊怪
- 在初期森林的南部有一个任务, 可以交杀死熊怪掉落的珠串来提高声望
