燃烧的远征升级指南 (58-70)
==============================================================================

从燃烧的远征起, 魔兽世界的任务链设置就有了非常大的变化. 通常任务都由各个地图的主要营地开始, 各个营地都有指引你前往其他营地的指引任务. 任务变得集中, 很少有经典旧世中跨越地图的任务了. 所以基本上是跟着任务链一直做即可.

- 地狱火半岛: 58-62
- 赞加沼泽: 62-64
- 泰罗卡森林: 64-66
- 纳格兰: 66-67
- 刀锋山: 67-68
- 虚空风暴: 68-70
- 影月谷: 68-70
