战斗属性::

    .aura 13845 精准: 使你的武器和毒药命中目标的几率提高5% (物理和法术都有效, 因为毒药是法术效果)

    .aura 20121 定罪: 使你的所有法术和攻击造成致命一击的几率提高5% (全职业全致命5%)
