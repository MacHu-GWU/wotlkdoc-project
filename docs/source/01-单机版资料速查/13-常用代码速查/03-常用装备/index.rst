传家宝套装
------------------------------------------------------------------------------
传家宝套装提供了以下几个部位的装备:

- 武器
- 饰品 x 2
- 戒指 x 1
- 肩膀
- 胸甲

~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

    # 武器
