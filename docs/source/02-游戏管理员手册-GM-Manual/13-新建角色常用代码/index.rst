需要从职业任务获得的技能
------------------------------------------------------------------------------
有些职业的重要技能需要做职业任务才能获得. 我们可以跳过这些繁琐的任务, 直接用GM命令习得这些技能.

**战士**::

    # 防御姿态
    .learn 71

    # 嘲讽
    .learn 355

    # 破甲
    .learn 7386

    # 狂暴姿态
    .learn 2458

**圣骑士**::

    # 救赎
    .learn 7328

**猎人**::

    # 驯服野兽
    .learn 1515

    # 喂食
    .learn 6991

    # 召唤宠物
    .learn 883

    # 解散宠物
    .learn 2641


**萨满**::

    # 石肤图腾
    .learn 8071

    # 灼热图腾
    .learn 3599

    # 治疗图腾
    .learn 5394


**德鲁伊**::

    # 熊形态
    .learn 5487

    # 巨熊形态
    .learn 9634

    # 猎豹形态
    .learn 768

    # 水栖形态
    .learn 1066

    # 旅行形态
    .learn 783

    # 飞行形态
    .learn 33943

    # 迅捷飞行形态
    .learn 40120

    # 低吼
    .learn 6795

    # 槌击
    .learn 6807


**术士**::

    # 召唤 虚空行者
    .learn 697

    # 召唤 魅魔
    .learn 712

    # 召唤 地狱猎犬
    .learn 691

    # 召唤 地狱火
    .learn 1122
