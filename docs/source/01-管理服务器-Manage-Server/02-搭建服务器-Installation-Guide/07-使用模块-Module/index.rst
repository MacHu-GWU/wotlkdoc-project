使用模块 Module
==============================================================================
AC 是模块化的项目. 模块定义了一些扩展功能, 每个模块相当于可插拔的功能, 类似于 Steam 上的游戏 Mod. 要使得这些模块在你的服务器上生效, 你需要在构建服务期之前, 把模块源码下载到 ``azerothcore-wotlk/modules/`` 目录, 然后再编译即可. 模块相关的配置会出现在 ``./bin/etc/modules`` 下.
