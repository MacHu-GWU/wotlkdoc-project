Color Code Example

- :gray:`gray`
- :gray:`gray`
- :silver:`silver`
- :white:`white`
- :maroon:`maroon`
- :red:`red`
- :magenta:`magenta`
- :fuchsia:`fuchsia`
- :pink:`pink`
- :orange:`orange`
- :yellow:`yellow`
- :lime:`lime`
- :green:`green`
- :olive:`olive`
- :teal:`teal`
- :cyan:`cyan`
- :aqua:`aqua`
- :blue:`blue`
- :navy:`navy`
- :purple:`purple`

- :under:`under`
- :over:`over`
- :blink:`blink`
- :line:`line`
- :strike:`strike`

- :it:`it`
- :ob:`ob`

- :small:`small`
- :large:`large`
