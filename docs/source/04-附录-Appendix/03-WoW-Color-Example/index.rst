WoW Code Example
==============================================================================
这里是一些 Color Code 的示例. 你可以用 ``:Color:`text``` 这样的语法在 RestructuredText 中书写带颜色的文字, 或者加入其他格式.

- :brown:`战士`
- :pink:`圣骑士`
- :red:`死亡骑士`
- :blue:`萨满`
- :darkseagreen:`猎人`
- :orange:`德鲁伊`
- :yellow:`盗贼`
- :blueviolet:`术士`
- :dodgerblue:`法师`
- :silver:`牧师`
